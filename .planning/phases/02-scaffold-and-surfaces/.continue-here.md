| Pattern | Description | Severity | Prevention Mechanism |

Eight systemic shape drifts only surfaced when live Paperclip hit them. | blocking | Handlers must import client types from `@paperclipai/plugin-sdk` (e.g. `PluginIssuesClient`, `PluginDatabaseClient`) and compose `Ctx` types from those. Integration-test layer in `test/worker/issue-reader-integration.test.mjs` pins one contract per drift; new handlers add equivalent integration tests. |
| Rapid SSH retry against fail2ban'd host | A burst of SSH attempts to Countermoves tripped fail2ban early in this session; subsequent calls timed out for ~30+ min. | advisory | For Countermoves SSH: one connection per task; if it times out, wait at least 15 min before retrying. If a handler's narrow Ctx differs from the SDK, that's the bug. |
