### `migrations/0017_loop_governor.sql` (migration, additive DDL)

**Analog:** `migrations/0016_reply_resume_dedup.sql` (read in full, 60 lines)

1. Fully-qualified namespace literal `plugin_clarity_pack_cdd6bda4bd.` on every table — **no template substitution**.
2. `CREATE TABLE IF NOT EXISTS` for idempotent re-install.
3. **Inline `UNIQUE (...)` inside `CREATE TABLE`** — NOT a standalone `CREATE INDEX` (the host `extractQualifiedRefs` rejects standalone `CREATE INDEX ... ON schema.table`). The inline UNIQUE produces the btree the lookup needs.

5. Statements begin with `create` / `alter` / `comment` only — no `DO $$`, no procedural blocks.

**CREATE-TABLE pattern to mirror** (`0016` lines 47-59 — column conventions: `id bigserial PRIMARY KEY`, `company_id text NOT NULL`, `created_at timestamptz NOT NULL DEFAULT now()`, `<flag> boolean NOT NULL DEFAULT false`, inline `UNIQUE`, apostrophe-free `COMMENT ON`):

  agentKey: string,

         FROM plugin_clarity_pack_cdd6bda4bd.editor_agent_failures

## Shared Patterns

### Durable-table repo ctx contract
**Source:** `src/worker/db/reply-resume-repo.ts:1-23,55-84`
**Apply to:** all three new repos (`own-operation-issues`, `wake-ledger`, `wake-kill-switch`)
- `ctx.db.query<T>(sql, params)` — SELECT-only, single statement, returns rows.
- `ctx.db.execute(sql, params)` — namespace DML, returns `{rowCount}` only, **NEVER rows / no `RETURNING`** (L-9). A read-back is a separate `query`.
- Idempotency via `ON CONFLICT (...) DO NOTHING` / `DO UPDATE` at the repo INSERT layer, not the migration.
- All params parameterized `$1`/`$2` — no string interpolation of identifiers (V5 input validation).

### Manual-clear governance parity
**Source:** `src/worker/agents/circuit-breaker.ts:100-114`
**Apply to:** `wake-governor.ts` + `wake-kill-switch-repo.ts`
- Engage automatically once at threshold; **never auto-clear** (clearing is an operator gesture — coexistence #4). Fail-open on durable-read errors.

### Version-scoped durable state
**Source:** `src/worker/agents/circuit-breaker.ts:24,38,164-183`
